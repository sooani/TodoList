# TodoList



## 간단한 TodoList

- **할일 등록**: 할일 목록 등록
- **할일 수정**: 내용 수정, 할일 완료 여부
- **할일 조회**: 전체 할일 조회, 특정 할일 조회
- **할일 삭제**: 전체 할일 삭제, 특정 할일 삭제


## 추가 설정

- **Java 버전**: Java 11.
- **빌드 도구**: Gradle.
- **코딩 및 문서 인코딩**: UTF-8.
- **Javadoc 설정**: 프로젝트의 모든 Java 소스에 대한 Javadoc 생성 지원.
- **의존성 관리**: Spring Dependency Management 플러그인을 통한 의존성 버전 관리.
